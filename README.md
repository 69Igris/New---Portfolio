# Modern Full-Stack Portfolio Website

A stunning, modern personal portfolio website built with React, Node.js, and cutting-edge web technologies. Features cinematic animations, glassmorphism UI, and interactive graphics.

## 🚀 Features

### Frontend
- **React 18** with modern hooks and functional components
- **Framer Motion & GSAP** for cinematic scroll-triggered and interactive animations
- **Interactive Graphics** customized per-page using Lucide React icons
- **Tailwind CSS** for responsive, utility-first styling and glassmorphism
- **Space-themed background** with moving stars and particles
- **Custom cursor** with neon glow effects
- **Typewriter effect** for dynamic text animations
- **Responsive design** for all devices
- **Smooth page transitions** and navigation

### Backend
- **Node.js** with Express.js
- **MongoDB** for data persistence
- **RESTful API** endpoints
- **Email integration** with Nodemailer
- **Contact form** with backend processing
- **Project management** system

### Pages
1. **Home Page** - Hero section with cinematic animations and interactive hero graphics
2. **Skills Page** - Animated skill grid with progress bars and tech visualizations
3. **Projects Page** - Interactive project cards with seamless showcase graphics
4. **Contact Page** - Contact form with backend integration and connect graphics

## 🛠️ Tech Stack

### Frontend

## 📦 Installation

### Prerequisites
- Node.js (v16 or higher)
- MongoDB (local or cloud)

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/69Igris/New---Portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm run install-all
   ```

3. **Set up environment variables**
   ```bash
   cd server
   cp env.example .env
   ```
   
   Edit `.env` file with your configuration.

4. **Start the development servers**
   ```bash
   npm run dev
   ```
   This concurrent script will start both the frontend and backend sequentially.
